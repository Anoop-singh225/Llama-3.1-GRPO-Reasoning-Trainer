# Llama-3.1-GRPO-Reasoning-Trainer
Fine-tuning Llama-3.1 using Group Relative Policy Optimization (GRPO) to enhance mathematical reasoning and logical deduction. This repository contains the training pipeline, custom reward functions (XML, Format, Correctness), and LoRA adapters from the 128-step checkpoint. Developed as part of the NextMatrix Tech ecosystem for advanced AI solution
